Partial-Fraction Expansion (部分分式展開)

假設F(s)\mathcal{F}(s)F(s)長得很複雜，複雜到難以直接計算拉普拉斯轉換
那麼將F(s)\mathcal{F}(s)F(s)拆分成
F(s)=簡單項+簡單項+簡單項......\mathcal{F}(s) = 簡單項 + 簡單項 + 簡單項 ......F(s)=簡單項+簡單項+簡單項......的形式，再逐項進行拉式轉換
就會容易得多
(拉式轉換性質：L[f1+f2]=L[f1]+L[f2]\mathcal{L}[f_1 + f_2] = \mathcal{L}[f_1] + \mathcal{L}[f_2]L[f1​+f2​]=L[f1​]+L[f2​])

那如果像這個結構 F(s)=N(s)D(s)\mathcal{F}(s) = \frac{\mathcal{N}(s)}{\mathcal{D}(s)}F(s)=D(s)N(s)​
其中分子的階數 ≥\ge≥ 分母階數的話

Ex. F(s)=s3+2s2+6s+7s2+s+5\mathcal{F}(s) = \frac{s^3 + 2s^2 + 6s + 7}{s^2 + s + 5}F(s)=s2+s+5s3+2s2+6s+7​

2.3 The Transfer Function (轉移函數)

一言以蔽之：0初始條件、0平衡點下，輸入與輸出的關係式
G(s)=OutputInput\it{G(s)} = \frac{Output}{Input}G(s)=InputOutput​

初始條件：輸入為0、時間在0以前 的恆定輸出

給定一個系統(以方程式來描述其系統行為)
定義其中的 某項為輸入、某項為輸出，初始條件設為000
求 輸出輸入\frac{輸出}{輸入}輸入輸出​ 的樣子的過程，就是在求轉移函數
最精簡的例子(只是直觀理解，實際上不應該用此例)：
y=f(x)=2x+3y = f(x) = 2x + 3y=f(x)=2x+3
定義 xxx 為輸入，yyy 為輸出，那 333 則為初始條件
則轉移函數為 yx=2\frac{y}{x} = 2xy​=2
